# معرض الغواصات — مجلد كامل جاهز

هذا المشروع ثابت (Static) يعرض غواصات مع سلة شراء محلية ووظيفة "تواصل للخاص" عند الدفع.

محتويات المجلد:
- index.html  : الصفحة الرئيسية (عرض المنتجات، سلة الشراء، مودالات)
- styles.css  : تنسيقات الموقع (RTL، عربي)
- script.js   : وظائف السلة، التخزين المحلي، ونافذة التواصل
- images/     : ملفات SVG لصورة الغواصات (مضمنة كنماذج)

التشغيل محليًا:
1. أنشئ مجلدًا وضع الملفات كما هي (index.html, styles.css, script.js, images/).
2. افتح index.html في المتصفح.
3. لإرسال الطلب الفعلي، عند الضغط على "الدفع / تواصل للخاص" ستظهر نافذة تحتوي على رسالة جاهزة يمكنك نسخها وإرسالها للبائع في الخاص.

ملاحظة قانونية:
- هذا مشروع تعليمي/عرض فقط. لا يُشجَّع أو يُقدّم أي محتوى يخالف القوانين أو شروط الألعاب.
- الصور المستخدمة من نوع SVG مضمّن وموارد احتياطية من Unsplash (روابط مباشرة) لغرض العرض فقط.
